# Sistema de Ventas en JavaScript

## Descripción

Este proyecto es un **sistema de ventas básico** desarrollado en JavaScript para Node.js.  
Permite registrar productos con precio y stock, calcular el precio total incluyendo IVA, aplicar descuentos y actualizar automáticamente el inventario.

Está diseñado para **aprender y practicar** conceptos fundamentales de JavaScript como:

- Objetos
- Funciones
- Operadores y cálculos matemáticos
- Condicionales (`if`, `else`)
- Manejo de datos dinámicos (stock y precios)

Este sistema simula un flujo de ventas real en una empresa pequeña y puede servir como base para proyectos más avanzados.

---

## Funcionalidades

- Registrar productos con nombre, precio y stock.
- Calcular el precio total incluyendo IVA (16%).
- Aplicar descuentos por venta.
- Verificar y actualizar el stock automáticamente.
- Mostrar mensajes claros de confirmación de venta o aviso de falta de stock.

---

## Tecnologías utilizadas

- JavaScript (ES6+)
- Node.js
- Terminal / Consola para ejecución
